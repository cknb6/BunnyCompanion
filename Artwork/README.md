# 角色素材说明

角色为依据用户提供照片提炼的 2.5D 日系 Q 版形象。保留的视觉锚点包括长棕色波浪发、粉色兔子发夹、柔和五官、白粉配色服装和兔子元素。

- `ChromaSheets`：生成时的纯绿色背景原始动作表。
- `TransparentSheets`：去除绿色背景后的完整动作表。
- `sprite_manifest.json`：48 个最终精灵与原动作表单元格的对应关系。
- `../BunnyCompanion/Assets/Sprites`：程序实际嵌入的标准化透明 PNG。

参考照片原图未放入工程，避免在发送源码或 EXE 时额外传播私人照片。

这些角色素材仅用于本次私人桌宠礼物及其后续个人修改，不作为公共人物肖像或商业授权素材。

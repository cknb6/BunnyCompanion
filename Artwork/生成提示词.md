Do not create a photorealistic copy.
```text
Input image: the attached photo is an identity and design reference only.
Subject design: full-body petite chibi proportions, about 3.5 heads tall, tasteful ivory-pink short cardigan over a white pleated dress, white ankle socks, soft pink indoor shoes, tiny bunny charm bracelet; elegant and cute rather than childish.
Composition: a clean four-view model sheet in a 2x2 layout: front standing, 3/4 view, side view, back view. Exactly one character in each quadrant, full body fully visible, same scale, same outfit, same face, same hair and accessories. Generous spacing between views. No overlaps.
Rendering: polished 2.5D anime game asset, soft cel shading with subtle dimensionality, crisp silhouette, production-quality edges, consistent neutral studio illumination.
Background: perfectly flat solid #00ff00 chroma-key background for background removal; one uniform color, no grid, no shadows, no gradients, no texture, no floor plane, no reflections.
Constraints: no text, no labels, no watermark, no decorative border, no props, no extra limbs or fingers, no cropped body parts. Do not use #00ff00 anywhere in the character.
```

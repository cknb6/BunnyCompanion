小申陪伴 1.1
====================

一、运行

双击 EXE 即可（自包含打包，约 100MB 级，**无需安装 .NET**）。
第一次运行会加入当前用户开机启动项，不需要管理员权限。

如果 Windows SmartScreen 提示“Windows 已保护你的电脑”，这是个人制作、未购买商业
代码签名证书的程序常见提示。确认文件来自你信任的人后，可点击“更多信息—仍要运行”。

推荐从 GitHub Releases 下载最新版：
https://github.com/cknb6/BunnyCompanion/releases

请按电脑选择（每个架构只要下一个）：
- 普通 Intel/AMD 电脑 → BunnyCompanion-win-x64.exe（或 小申陪伴-x64.exe）
- 原生 ARM64 Windows → BunnyCompanion-win-arm64.exe（或 小申陪伴-arm64.exe）
不确定时优先下 x64。

二、互动

1. 单击角色：按点击区域反馈（头/身体/脚底），短时间连点有彩蛋。
2. 双击角色：比心和亲亲。
3. 点击头部区域：摸头反馈。
4. 中键点击角色：打开聊天窗口（多模态 Agent，可看桌面/写代码/陪聊）。
5. 按住角色拖动：移动到任意位置。
6. 右键角色：打开完整功能菜单。
7. 左键单击右下角托盘图标：显示或隐藏桌宠。

三、快捷键（与托盘一致）

Ctrl+Shift+S　显示 / 隐藏桌宠
Ctrl+Shift+C　打开聊天
Ctrl+Shift+P　切换鼠标穿透
Ctrl+Shift+,　个性化设置
Ctrl+Shift+H　快捷键说明

四、主要功能

透明无边框、置顶显示、启动入场动画、自动散步、48 个动作素材、粉嫩对话气泡、
多模态 Agent 对话（step-3.7 → OpenRouter 免费 → 本地中文兜底）、系统托盘、角色缩放、
鼠标穿透、多显示器边界保护、全屏自动隐藏、喝水提醒、休息提醒、
25 分钟专注陪伴、生日与纪念日彩蛋、安静时段、提示音、互动次数与爱心值记录。

角色周围的透明空白区域会自动穿透，不会形成一个挡住桌面点击的矩形框。

五、开机启动

第一次运行默认开启。若要关闭：
右键桌宠或托盘图标 → 取消勾选“开机自动启动”。

程序使用 Windows 当前用户启动项，不写系统目录，不请求管理员权限。
如果以后移动或重命名 EXE，请重新双击运行一次，启动项会自动更新为新路径。

六、个性化

右键 → 个性化设置，可以修改：

- 她的称呼和桌宠名字
- 角色大小、置顶、自动散步、声音和全屏隐藏
- 喝水及休息提醒间隔
- 安静时段
- 生日与纪念日
- 每行一句的专属对话

对话中可使用 {name} 代表她的称呼，{pet} 代表桌宠名字。

七、彻底退出或卸载

彻底退出：右键托盘图标 → 完全退出。

卸载：先取消“开机自动启动”并完全退出，再删除“小申陪伴.exe”。
本地设置位于：%LocalAppData%\BunnyCompanion
如需清除所有记录，可手动删除该文件夹。

八、隐私

照片原图没有打包进程序；程序只包含由参考图衍生的 Q 版动作素材。
姓名、纪念日、互动次数和位置均只保存在本机。
基础陪伴与本地台词库可完全离线；打开 Agent 聊天并走在线通道时才会联网。
看桌面需要你主动点「看桌面」或相关意图，不会后台偷偷连环截屏。
